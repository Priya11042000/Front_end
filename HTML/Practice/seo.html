Doctype :

The doctype tells the browser which version of HTML the webpage is using.
If the doctype is not given, the browser renders in quirks mode.
If the doctype is given, it renders in standards mode.

  • HTML (Old): Doctype is long and complex.
  • HTML5: Doctype is short and simple â†’ <!DOCTYPE html>

Lang Attribute :

The lang attribute is used to tell the browser which language the content is written in.
Example: <html lang="en">

Meta Tags :

Meta tags are placed inside the <head> section.
They provide information about the webpage to browsers and search engines.
Uses: SEO, social sharing, responsive design.

  • Charset : Defines encoding of the page.
    Example: <meta charset="UTF-8">
  • Viewport : Makes webpage responsive.
    Example: <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Description : Provides a summary of the webpage.
    Example: <meta name="description" content="Relevant content of webpage">
  • Keywords : Defines keywords of the page (less used today).
    Example: <meta name="keywords" content="HTML, CSS, Web Development">
  • Author : Defines the author of the page.
    Example: <meta name="author" content="Your Name">

Social Media Meta Tags :

Open Graph (for Facebook, WhatsApp, Instagram)

<meta property="og:title" content="Page Title">
<meta property="og:description" content="Page Description">
<meta property="og:image" content="image.jpg">
<meta property="og:url" content="https://example.com">

Twitter Meta Tags

<meta name="twitter:title" content="Page Title">
<meta name="twitter:description" content="Page Description">
<meta name="twitter:image" content="image.jpg">
<meta name="twitter:url" content="https://example.com">

Difference Between HTML & HTML5 :

Feature HTML (Old Versions) HTML5 (Latest)
Doctype Long & Complex Simple: <!DOCTYPE html>
Media Needed plugins like Flash Built-in support with <audio> and <video>
Semantics Mostly used <div> Semantic tags (<header>, <footer>, <article>)
Storage Cookies only Local Storage & Session Storage
Graphics No native graphics <canvas> and SVG supported
Mobile Support Limited Responsive with viewport
