Description of Semantic tag :

Semantic is used to give the meaning to both the developer and the browser to understand the structure of the webpage. we used some tag to structure to show what content is given in the tag and which tag content has to show 1st and arrange.

  • HEADER : is used to show the tag content has to come in top of the page.
  • MAIN : is used to show the tag content is main content to user and browser.
  • NAV : is used to show the tag content to navigate the page from one tab to another tab or with the same page.
  • SECTION : is used to show the tag content has to come in the section format.
  • ARTICAL : is used to show the tag content will be contented oriented information .
  • ASIDE : is used to show the tag content is has come as a slider , it can be from left side or right side.
  • FOOTER : is used to show the tag content is has to come in bottom of the page
