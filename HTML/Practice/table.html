Description of Table tags :

A table is used to display data in rows and columns.

Important tags in table :

  1. < table > :   Defines a table.
  2. < thead > :   Table header section (usually column headings).
  3. < th > :   Table header (bold & centered by default)
  4. < tbody > :   Table body section (main data rows).
  5. < tr > :   Table row.
  6. < td > :   Table data.
  7. < tfoot > :   Table footer section, e.g : totals.

Extra Attributes for <table >

  • border="1" → Adds border
  • cellpadding → Space inside cells
  • cellspacing → Space between cells
  • colspan → Merge columns
  • rowspan → Merge rows
